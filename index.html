Instructions:

Clique pour commencer. WASD pour avancer, espace pour sauter, clic gauche pour casser.

Shift + clic: poser un bloc. Touches 1-9: changer le bloc sélectionné.

E: ouvrir/fermer l'inventaire. M: changer de mode (survie/créatif). D: changer difficulté.

Clic droit: utiliser/eat si nourriture.

Échap pour relâcher la souris.
